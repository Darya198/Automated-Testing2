# Проект автоматизации тестирования (Урок 10)

В данном проекте реализованы автотесты для медленного калькулятора и интернет-магазина SauceDemo с использованием паттерна **Page Object Model**, библиотек **Selenium**, **Pytest** и отчетности **Allure**.

## Особенности проекта
- Все методы классов задокументированы с использованием **Type Hinting** (аннотации типов).
- Тесты размечены шагами `@allure.step` и метаданными (`feature`, `story`, `severity`).
- Реализована автоматическая очистка данных в тестах.

## Запуск тестов и формирование отчета

1. **Запуск тестов калькулятора:**
   `pytest calc_test.py --alluredir=allure-results`

2. **Запуск тестов интернет-магазина:**
   `pytest shop_test.py --alluredir=allure-results`

## Просмотр отчета Allure

Для генерации и автоматического открытия отчета в браузере выполните команду:
`& "$env:USERPROFILE\scoop\apps\allure\current\bin\allure.bat" serve allure-results`

*Примечание: Для корректной работы Allure в системе должна быть установлена Java (JDK 17 или выше).*
